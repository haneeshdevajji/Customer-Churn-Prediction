## 🐛 TROUBLESHOOTING

### Issue: Data file not found
```
FileNotFoundError: data/WA_Fn-UseC_-Telco-Customer-Churn.csv
```
**Solution:** Ensure CSV is in the `data/` folder

### Issue: Library import errors
```
ModuleNotFoundError: No module named 'sklearn'
```
**Solution:** Run `pip install -r requirements.txt`

### Issue: Low recall on churn
```
Recall: 0.3 (only catching 30% of churners)
```
**Solution:** Decrease threshold (e.g., from 0.7 to 0.5)

### Issue: Images folder not created
```
FileNotFoundError: [Errno 2] No such file or directory: 'images'
```
**Solution:** Code auto-creates it with `os.makedirs("images", exist_ok=True)`
